## Usage
Download the package with git

    git clone git@github.com:purple4reina/MazeGenerator.git
    cd MazeGenerator

Compile

    javac MazeGenerator.java

Run

    java MazeGenerator 20 10  # maze with width 20 and height 10
    
Enjoy :clap:
